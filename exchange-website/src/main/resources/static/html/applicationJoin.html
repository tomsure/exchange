申请成为保荐商

申请须知

为了使得ACAE平台会员的资质得到进一步的保证，所有申请会员的企业必须聘请ACAE认可的保荐商。保荐商在企业申请会员过程中起到主导的作用，对企业进行尽职调查、为企业对接所需第三方机构、为企业递交会员申请材料等。

ACAE可全权酌情拒绝任何作为保荐商的申请，尽管申请人符合本申请所列的资格标准。任何符合资格条件并经ACAE批准的申请人，必须签署“作为ACAE保荐人协议”(以下简称“协议”)。一旦签署该协议，申请人将被列入获批准的ACAE保荐商名单，该名单已在ACAE网站上公布（网址）。经批准的ACAE保荐商必须由企业选择作为其会员申请保荐人。ACAE会员的职责在《ACEA会员申请规则》中作了说明。《ACAE会员申请规则》也在ACAE网站上发布。

请注意，保荐商申请费用为XXX元，并须连同填妥的付款表格(附于本申请表格末尾)一并递交，否则申请将不获处理。不论申请是否获批准、拒绝或撤回，申请费用均不获退还。填妥、签署的申请表和所有必需的补充材料应通过电子邮件发送至sponsor@acae.com，或邮寄至ACAE办公地址。

任何ACAE保荐商还必须支付年费，该年费与申请费分开，并在每年12月为下一个日历年度计费。ACAE可自行决定不时调整年费。目前的年费由ACAE在其网站中公布

    保荐商职责

  • 为企业提交ACAE申请会员介绍信
  • 对公司进行专业全面的尽职调查
  • 负责递交企业成为会员的首次信息披露
  • 在与管理层建立长期关系时，为企业创造投资者需求提供专业指导
  • 协助公司识别对市场重要的信息，并向投资者披露
  • 建议企业如何战略性地提高投资者对公司的兴趣
  • 教育企业使其了解其成为ACAE会员应尽的责任

    申请资质

  1. 申请人有足够为企业提供服务的经验，能够根据《ACAE会员申请规则》更好的帮助企业，同时能够很好地教育、劝告和协助企业履行其披露义务。
  2. 申请人目前没有受到任何来自任何证券监管机构或任何自我管理机构的纪律处分的制裁。
  3. 若申请人可以为企业提供申请会员所需的其他工作（审计、评估及架构搭建等），申请人需提交相关资质证明。
