为了使得ACAE平台会员的资质得到进一步的保证，所有申请会员的企业必须聘请ACAE认可的保荐商。保荐商在企业申请会员过程中起到主导的作用，
对企业进行尽职调查、为企业对接所需第三方机构、为企业递交会员申请材料等。
ACAE可全权酌情拒绝任何作为保荐商的申请，尽管申请人符合本申请所列的资格标准。任何符合资格条件并经ACAE批准的申请人，必须签署“作为ACAE保荐人协议”(以下简称“协议”)。
一旦签署该协议，申请人将被列入获批准的ACAE保荐商名单，该名单已在ACAE网站上公布（网址）。经批准的ACAE保荐商必须由企业选择作为其会员申请保荐人。
ACAE会员的职责在《ACEA会员申请规则》中作了说明。《ACAE会员申请规则》也在ACAE网站上发布。 请注意，保荐商申请费用为XXX元，并须连同填妥的付款表格(附于本申请表格末尾)一并递交，否则申请将不获处理。
不论申请是否获批准、拒绝或撤回，申请费用均不获退还。填妥、签署的申请表和所有必需的补充材料应通过电子邮件发送至sponsor@acae.com，或邮寄至ACAE办公地址。
任何ACAE保荐商还必须支付年费，该年费与申请费分开，并在每年12月为下一个日历年度计费。ACAE可自行决定不时调整年费。目前的年费由ACAE在其网站中公布
